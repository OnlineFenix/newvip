HOW TO USE APP

➨ Uploading Messages

Is app me aapko 2 input box dikhayi denge aap apne game ke message ko un dono box me enter kar sakte hai aapka message kis type ka hai ye decide karega ki aapko apna game konse box me likhna hai.

Single Line Message : Agar aapka message sirf ek line me hai, toh seedha type karein aur enter karein.

Multiple Lines Message : Agar aapka message multiple lines me hai, toh har line ko enter se alag karein. Isse system asani se har line ko process kar sakega.

Harup Messages : App apni game me haruf bhi likh sakte hai jo hamari app automatically detect kar legi jise detect karne ke kuch rules hai jo harup detect section me likhe hain.

Crossing Messages : Agar aapki game ek crossing hai jo ki "123456(50)" or "21 22 23 24 25 ...... = 50 into" is type ki hai to apni game ko dusre number ke input box me enter karen.

➨ Submission and Visualisation

Submit Messages : Aap apne messages ko input box me likh kar submit button par click kare uske baad data ka process background me hoga par yah itna kam time lega ki aapko submit karne ke baad koi wait nahi karna hoga.

Visualisation : Aap apne message ko submit krne ke baad jo numbers aur harup ki table hai usme apna data dekh sakte hai iske liye aapko koi alag process ki jarurat nahi padegi submit karte hi aapka data table me visible ho jayega.

Understanding : Aap jo message input karenge usme se numbers aur amount ko detect karke jo numbers hai unka amount table me save kar diya jayega agar harup hai to harup ki table me data visible ho jayega.

Quick Results : Aap 'Quick Check' button par click karke aasani se pata kar sakte hai ki konse number ya harup par minimum aur maximum amount save kiya gaya hai aur apne numbers aur harup par total kitna amount save hai.

FEATURES AND ADVANTAGES

➨ Features

Numbers and Amount Detection : Aap jo bhi messages input karenge yeh app un messages me numbers aur amount ko detect karega aur table me visible karayega.

Harup Detection : Ye app aapke enter kiye gaye message me se harup ko detect karke unko harup ki alag table me visible karwayega par yaad rahe jab bhi aap koi message enter karen aur usme harup likhe ho to wo harup alag line me hone chahiye aur harup '888ab=50', 'A333(150)' ya '111-666ab=100' is type se likhe hone chahiye.

Crossing Detection : Agar aapki game ek crossing hai to dhyan rahe ki use aap crossing ke input box me hi enter karen ye app aapki crossing me se numbers aur amount ko aasani se detect kar lega isme aap '1234567(100)', '98765=25into' ya '12 13 14 15 16 ... (20 into)' is type ki crossing enter krke table me data save kar sakte hain.

Copy and Share : Aap apni table ko text format me copy kar sakte hai aur kisi bhi social media platform par direct share bhi kar sakte hai jab aap copy button pr click karenge to aapki table aapke clipboard me copy ho jayegi aur share button par click karke aap bina copy kiye direct share bhi kar sakte hain.

Disadvantages : Agar aapke message me signs aur symbols 2 ya usse jyada ek hi jagah pr hai to kosis kare ki aap double signs ko remove karde ye app apni taraf se sare double signs ko delete kar deta hai pr aap agar iska dhyan rakhenge to aapko desired result dekhne ko milega.

➨ User Interface

Intuitive Interface : App ka interface aasan aur user friendly hai. Aap bina kisi technical knowledge ke is app ko aasani se intemal kar sakte hain.
